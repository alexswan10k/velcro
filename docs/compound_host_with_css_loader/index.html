Simple React with css using the webpack css-loader

This example is similar to the compound host with caching example, but adds some styling to the rendered markup using a css module loaded via the style-loader and css-loader webpack loaders.

The colours and styles added to the code below come from a dynamically compiled stylesheet read from the in-memory filesystem attached to the compound file system.

Remember, there is no server-side bundling happening here. Everything needed to execute this demo was loaded dynamically from unpkg.com. That is also why this take a while for the first load.
